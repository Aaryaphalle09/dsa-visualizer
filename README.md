# 🔍 DSA Visualizer (Python + Tkinter)

An interactive desktop application to visualize Data Structures and Algorithms step-by-step.

Built with Python and Tkinter — no external libraries needed!

---

## ✨ Features

### Sorting Algorithms
- **Bubble Sort** — step-by-step comparisons and swaps
- **Selection Sort** — minimum element selection visualized
- **Insertion Sort** — insertion into sorted portion shown live

Each sorting visualization shows:
- Live comparison and swap counters
- Color-coded bars (comparing, swapping, sorted)
- Step-by-step explanation text
- Time & Space complexity display
- Adjustable array size and animation speed

### Graph Traversal
- **BFS (Breadth-First Search)** — queue-based level order traversal
- **DFS (Depth-First Search)** — stack-based deep traversal

Graph features:
- Click to add nodes on canvas
- Click two nodes to connect with edge
- Color-coded states (unvisited, in-queue, visited)
- Step-by-step traversal log
- Sample graph for quick demo

## 🚀 How to Run

### Requirements
- Python 3.x (Tkinter is built-in — no pip install needed!)

### Run the app
```bash
# Clone the repo
git clone https://github.com/Aaryaphalle09/dsa-visualizer.git
cd dsa-visualizer

# Run directly
python dsa_visualizer.py
```

On Windows you can also just double-click `dsa_visualizer.py`!

## 📸 How to Use

**Sorting:**
1. Select algorithm from dropdown
2. Adjust array size and speed
3. Click **Generate Array**
4. Click **▶ Sort** to start

**Graph:**
1. Click **Load Sample** for quick demo
2. Or use **+ Add Node** mode → click canvas to place nodes
3. Switch to **↔ Add Edge** → click two nodes to connect
4. Click **▶ BFS** or **▶ DFS** to visualize traversal
