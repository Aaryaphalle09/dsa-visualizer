# 🔍 DSA Visualizer

An interactive web-based tool to visualize Data Structures and Algorithms step-by-step.

🔗 **Live Demo:** https://aaryaphalle09.github.io/dsa-visualizer

---

## ✨ Features

### Sorting Algorithms
- **Bubble Sort** — step-by-step comparisons and swaps
- **Selection Sort** — minimum element selection visualized
- **Insertion Sort** — insertion into sorted portion shown live

Each sorting visualization shows:
- Live comparison and swap counters
- Color-coded bars (comparing, swapping, sorted)
- Step-by-step explanation of what's happening
- Time and Space complexity display
- Adjustable array size and animation speed

### Graph Traversal
- **BFS (Breadth-First Search)** — queue-based level traversal
- **DFS (Depth-First Search)** — stack-based deep traversal

Graph features:
- Click to add nodes anywhere on canvas
- Connect nodes by clicking two nodes in Edge mode
- Color-coded node states (unvisited, in-queue, visited)
- Step-by-step traversal log
- Sample graph loader for quick demo

```bash
cd dsa-visualizer

```

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- HTML5 Canvas API (for graph visualization)
- No frameworks, no dependencies — runs entirely in browser

---

## 📸 How to Use

**Sorting:**
1. Select an algorithm from the dropdown
2. Adjust array size and speed sliders
3. Click "Generate Array"
4. Click "▶ Sort" to start visualization

**Graph:**
1. Click "Load Sample Graph" for a quick demo
2. Or click canvas in "Add Node" mode to place nodes
3. Switch to "Add Edge" mode and click two nodes to connect
4. Click "▶ BFS" or "▶ DFS" to start traversal
